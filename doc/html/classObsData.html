Function to perform preprocessing.

This function performs preprocessing by filling gaps using lagrange interpolation and removing phase jumps using quartiles and Inter Quartile Range.

Parameters
minArcLenminimum data duration(Seconds) to consider an arc valid.
intrpolIntrvlMaximum gap duration (Seconds) to interpolate.
degDegree of Interpolation, passed to lagrangeInterpolation.
