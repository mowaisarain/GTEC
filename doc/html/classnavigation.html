This Function apply rotations around uk, ik and Lk.

This Function apply rotations around uk, ik and Lk, Rotation == | Xk | | rk | | Yk | = R3(-Lk)R1(-ik)R3(-uk) | 0 | | Zk | | 0 | Wwhere R1 and R3 are the rotation matrices defined at: http://www.navipedia.net/index.php/Transformation_between_Terrestrial_Frames By Hernández-Pajares, Technical University of Catalonia, Spain.

Function to compute eccentricity anomaly Ek.

This Function computes eccentricity anomaly Ek by Solving (iteratively) the Kepler equation for the eccentricity anomaly, using Newton–Raphson method, Equation –> Mk = Ek - ( e * Sin(Ek) )
