## Welcome to GTEC GitHub Repository

GTEC is a free software written in C++ (Standard C++11) and distributed under GPL version 2.0 license. GTEC implements multi constellation GNSS derived TEC(Total Electron Content) calibration model originaly proposed by _Luigi Ciraolo_[[^^]](#ref1) with multi constellation extension by Katy Alazo Cuartas[[^^^]](#ref2) from [T/ICT4D Lab](http://t-ict4d.ictp.it/nequick2/gnss-tec-calibration) ICTP - Abdus Salam International Center for Theoretical Physics.


### Web Model
A web based model is also available at [T/ICT4D GNSS TEC Calibration](http://t-ict4d.ictp.it/nequick2/gps-tec-calibration-online)
